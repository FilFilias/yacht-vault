# Booking Model

**Summary**: How renters discover, compare, and book yachts on YachtBay.

---

## Booking Type: Instant Book

Renters can confirm a booking immediately — no owner approval step required. The booking is confirmed as soon as payment is processed.

**Why instant book:**
- Reduces friction and eliminates back-and-forth delays
- Higher conversion — renters don't drop off waiting for a response
- Matches expectations set by platforms like Airbnb and GetMyBoat

See [[decisions/2026-05-01-instant-booking]].

## Charter Duration Options

Both short and long charters are supported:

| Type | Description |
|------|-------------|
| **Day trip** | Single-day booking |
| **Multi-day** | 2–6 consecutive days |
| **Weekly charter** | 7-day booking (standard in the industry) |

Pricing is set by the owner per duration type.

## Renter Journey

1. Browse available yachts (search by location, date, duration, type, crew option)
2. Compare yachts — photos, features, specs, pricing, reviews
3. Select crew configuration (if multiple options available)
4. Instant book — confirm dates and pay
5. Receive booking confirmation and owner contact details

## Owner Responsibilities

- Set availability calendar
- Define pricing per charter type
- Define available crew configurations
- Respond to any post-booking queries
