# Roadmap Overview

**Summary**: High-level phases for building and growing YachtBay.

---

> [!note]
> This is a living document. Phases and scope will be refined as the product is defined.

## Phase 1 — MVP

**Goal**: Validate the core marketplace loop in Greece.
Core loop: owner lists a yacht → renter finds and instant-books it → transaction completes.

See [[specs/mvp-scope]] for the full feature list.
See [[roadmap/phase-1-development]] for the 7-milestone build order.

## Phase 2 — Growth

**Goal**: Grow supply (more owners/yachts) and demand (more renters/bookings).

Likely additions:
- Reviews and ratings system
- Owner dashboard with analytics
- Calendar sync (iCal / Google Calendar)
- Promotional tools for owners

## Phase 3 — Expansion

**Goal**: Expand to broader Mediterranean (Croatia, Turkey, Italy, etc.)

Requires:
- Multi-currency support
- Multi-language (EN as base, add local languages)
- Legal/compliance per new market
