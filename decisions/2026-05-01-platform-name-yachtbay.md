# Temporary Platform Name: YachtBay

## Context

The platform needs a working name for documentation and development. A final brand name has not been decided yet.

## Options Considered

Several directions were explored:

| Name | Notes |
|------|-------|
| Poseidis | Greek/mythological feel |
| Aegis | Aegean + trust/protection |
| Nautica | Clean, internationally understood |
| YachtBay | Simple, descriptive, SEO-friendly |
| HarbourKey | Nautical, implies access |
| ChartHouse | Nautical term, implies expertise |
| BlueDeck | Clean, visual |
| Seakind | Sea + human warmth |

## Decision

**YachtBay** as a working name — simple, instantly understood, globally accessible, and good for SEO. This is explicitly temporary and will be revisited.

## Consequences

- Use "YachtBay" consistently throughout the vault and codebase for now
- When the final name is decided, update vault-wide (the CLAUDE.md rules note this explicitly)
- Domain availability and trademark checks should be done before committing to a final name
